# Aither — AI Health Scanner

An AI-powered health diagnostics tool that analyzes text input for emotional risk, self-harm signals, personality patterns, and tone. Built to make AI safety and mental health screening more accessible and interpretable.

## What it does

- Self-harm and crisis signal detection
- Emotional risk scoring
- Personality and tone analysis
- Ethical risk assessment
- Real-time visual diagnostics and reporting

## Stack

| Layer | Tech |
|---|---|
| Language | Python |
| Interface | Gradio |
| AI/ML | Hugging Face models |
| Deployment | Procfile (Heroku-compatible) |

## Run locally

```bash
pip install -r requirements.txt
python aither_baseline.py
```

## Background

Built as part of Aither — a broader AI health and diagnostics platform focused on human-centered AI interaction, interpretability, and user trust in AI-generated outputs.
